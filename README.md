# 🍕 Dish Finder (React)

A modern React app to search and explore meals using a public API. Users can search for dishes, view details, and save their favorite recipes.

---

## 🚀 Features

* 🔍 Search dishes and recipes in real-time
* 📋 View list of meals with images
* 📄 Click on a dish to see detailed information
* ❤️ Add and remove favorites (stored locally)
* ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

* React
* JavaScript (ES6+)
* CSS
* TheMealDB API

---

## 🌐 API Used

This project uses TheMealDB API:

* Search meals

  ```
  https://www.themealdb.com/api/json/v1/1/search.php?s=chicken
  ```

* Get meal details

  ```
  https://www.themealdb.com/api/json/v1/1/lookup.php?i=52772
  ```

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/nishant5788/react-dish-finder
cd react-dish-finder
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm start
```
